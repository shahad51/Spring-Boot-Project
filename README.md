# Product Management System

## 📌 Overview
A Product Management System developed using Spring Boot. The system allows users to manage products through CRUD operations, including creating, viewing, updating, and deleting products.

---

## 🚀 Features
- Add new products
- View product list
- Update product information
- Delete products
- RESTful API
- Database integration

---

## 🛠️ Technologies Used

- Java
- Spring Boot
- Spring Data JPA
- Maven
- MySQL
- REST API

---

## 📂 Project Structure

src/
├── controller
├── service
├── repository
├── entity
└── resources
